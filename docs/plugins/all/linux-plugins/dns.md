# linux_dns

### Description
- Active DNS server(s)


### Configuration
```ini
[linux_dns]
enabled = true
```

| Name | Options | Default | Description |
| ---- | ------- | ------- | ----------- |
| enabled | true/false | true | Plugin status |


### Returned values
```json
"linux_dns": [
    "8.8.8.8",
    "4.4.4.4",
]
```
