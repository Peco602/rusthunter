# linux_unusual_suid_root_files

## Description
This plugin provides a list with all SUID files on a Linux machine.
This requires knowledge of normal SUID files.
It is very useful if you are using a baseline.

## Return Values
Array of strings with only a value:

- *file 1 detail*
- *file 2 detail*
