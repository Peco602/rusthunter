# linux_root

## Description
This plugin provides a list of the root users on a Linux machine.

## Parameters
| Name | Options | Default | Description |
| ---- | ------- | ------- | ----------- |
| enabled | true/false | true | Plugin enable status |

## Return Values
Array of strings:

- *root1*
- *root2*
