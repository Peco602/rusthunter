# windows_users

## Description
This plugin provides a list of the available users on a Windows machine.

## Parameters
| Name | Options | Default | Description |
| ---- | ------- | ------- | ----------- |
| enabled | true/false | true | Plugin activation status |

## Return Values
Array of objects:

| Key | Description |
| --- | ----------- |
| Name | Username |
| Enabled | Status |
