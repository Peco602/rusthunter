# macos_users

## Description
This plugin provides a list of the available users on a macOS machine.

## Parameters
| Name | Options | Default | Description |
| ---- | ------- | ------- | ----------- |
| enabled | true/false | true | Plugin enable status |

## Return Values
Array of strings:

- *user1*
- *user2*
